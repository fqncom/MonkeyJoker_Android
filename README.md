# MonkeyJoker_Android
MonkeyJoker for android

## 项目工具及开发环境
* 该项目使用统一的keystore
* jdk版本为1.8及以上
* 使用AndroidStudio作为IDE

## 项目目录结构

 ### app
 * libs   ---jar包
 * src
  - assets
  - com
    - bean
    - ui   ---activity fragment
    - adapter
    - interface
    - listeners
    - view ---自定义view
    - tool ---各种工具类
    - util
    - data
    - api ---存放api连接

  - res
    - anim
    - layout
    - drawable ---存放自定义样式， .9文件，第三方资源文件
    - mipmap
    - values
    - menu
